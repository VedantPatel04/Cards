Pipeline Stages

1. Transaction Analysis (Blue) - Analyzes spending patterns and categorizes transactions
2. Card Validation Gate (Orange) - Validates card data quality with hard fails and soft warnings
3. Score Calculator (Purple) - Computes personalized scores for each card
4. Tie-Break Logic (Light Green) - Applies deterministic ranking rules
5. Confidence Check (Yellow) - Evaluates recommendation reliability
6. Output (Dark Green) - Returns top 3 recommendations with explanations
